* Basic ABAP program

REPORT z_tsi_demo.

* Main program logic
START-OF-SELECTION.
  WRITE: 'Hello, World! This is a basic ABAP program.'.

* Alternative simple approach
* WRITE: 'Hello, World! This is a basic ABAP program.'.

* Example with variables and types
* DATA: lv_name    TYPE string VALUE 'TSI Student',
*       lv_age     TYPE i VALUE 20,
*       lv_message TYPE string.
*
* lv_message = |Hello, { lv_name }! You are { lv_age } years old.|.
* WRITE: lv_message.

* Example function definition (using FORM)
* FORM greet USING p_name TYPE string
*              CHANGING p_greeting TYPE string.
*   p_greeting = |Hello, { p_name }!|.
* ENDFORM.

* Example usage
* DATA: lv_greeting TYPE string.
* PERFORM greet USING 'TSI Student' CHANGING lv_greeting.
* WRITE: lv_greeting.

* Example with internal tables and loops
* DATA: lt_numbers TYPE TABLE OF i,
*       lv_number  TYPE i.
*
* APPEND 1 TO lt_numbers.
* APPEND 2 TO lt_numbers.
* APPEND 3 TO lt_numbers.
* APPEND 4 TO lt_numbers.
* APPEND 5 TO lt_numbers.
*
* LOOP AT lt_numbers INTO lv_number.
*   WRITE: |Number: { lv_number }|.
* ENDLOOP.

* Example with structures
* TYPES: BEGIN OF ty_person,
*          name TYPE string,
*          age  TYPE i,
*        END OF ty_person.
*
* DATA: ls_person TYPE ty_person,
*       lt_persons TYPE TABLE OF ty_person.
*
* ls_person-name = 'TSI Student'.
* ls_person-age = 20.
* APPEND ls_person TO lt_persons.
*
* LOOP AT lt_persons INTO ls_person.
*   WRITE: |Hello, I'm { ls_person-name } and I'm { ls_person-age } years old!|.
* ENDLOOP.
